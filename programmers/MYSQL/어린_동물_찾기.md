## MySQL lv.1
동물 보호소에 들어온 동물 중 젊은 동물1의 아이디와 이름을 조회하는 SQL 문을 작성해주세요. 이때 결과는 아이디 순으로 조회해주세요.

```sql
SELECT ANIMAL_ID , NAME
FROM ANIMAL_INS
WHERE INTAKE_CONDITION != 'Aged'
ORDER BY ANIMAL_ID;
```
<aside>
❓ 이름이 없는 즉, NULL 값이 아닌 데이터의 경우를 구하는 것이므로 IS NOT NULL 을 사용하면 된다.
</aside>
